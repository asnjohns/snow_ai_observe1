# Snowflake AI Observability Experimentation

# AI Question Insights

The AI Question Insights framework provides comprehensive observability and analytics for Snowflake Cortex Analyst usage. This framework enables data teams to understand user behavior, optimize semantic models, identify patterns in natural language queries, and improve the overall AI agent experience.

# AI Success Feedback Analytics

The `AI_SUCCESS_FEEDBACK` schema provides comprehensive analytics and insights for Cortex Analyst request history, enabling data-driven improvements to AI/ML model performance and user satisfaction.

This schema works in conjunction with the `AI_QUESTION_INSIGHTS` schema, which contains the base `CORTEX_ANALYST_REQUEST_HISTORY` table.
